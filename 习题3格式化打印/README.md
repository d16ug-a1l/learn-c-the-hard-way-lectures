# 习题3格式化打印  

---

1. 我们先来解释一下代码,我直接加入一个嵌入式的代码块儿来解释,直接看每一行的注释即可,
```c
#include <stdio.h> //#是预处理指令的开头,include是包含一个文件到源文件中
                   //这个在预处理阶段会将stdio.h这个文件插入到源文件中
                   //标准库使用<>,自己定义的头文件使用""
int main(int argc, char* argv[]) //int是main函数的返回类型,main是主函数的入口
                                //argc是代表从终端中传入的参数数量,argv[]是一个字符串数组指针
                                //字符串数组指针存储的是我们传入的参数内容
{
    int age = 100;  //初始化age变量为100
    int height = 72;//和上面的一样

    printf("我 %d 岁了.", age);//%d是格式化输出,表示将传入的age变量将以整型的格式进行展示
   // printf("........"); //与上面的格式化输出一样

    return 0;
}
```  

---  

2. 现在我们修改这个程序,比如将age初始化为-100,或者将其没有初始化直接打印这个参数来看看效果,我在做练习的时候和作者是不一样的,我的系统是Ubuntu x64 24.04LTS版本,每次执行的结果都不一样,将age参数替换成argv再次查看执行情况,我这里的执行结果是一个已经数据溢出的数字,你看着是一个负数其实不是,只是数值太大了,符号位被溢出了,他其实是一个64位的内存地址.
