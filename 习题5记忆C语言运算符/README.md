# 习题5记忆C语言运算符
## 运算符列表
| 算术运算符 | 描述 |
| :---: | :---: |
| + | 加 |
| - | 减 |
| * | 乘 |
| / | 除 |
| % | 取模 |
| ++ | 自增 |
| -- | 自减 |  

|关系运算符 |描述|
| :---: | :---: |
|==|等于|
|!=|不等于|
|>|大于|
|<|小于|
|>=|大于等于|
|<=|小于等于|  

|逻辑运算符|描述|
|:---:|:---:|
|&&|逻辑与|
|\|\||逻辑或|
|!|逻辑非|
|?:|条件运算符、逻辑三元运算符|  

|按位运算符|描述|
|:---:|:---:|
|&|按位与|
|\||按位或|
|^|按位异或|
|~|按位取反|
|<<|按位左移|
|>>|按位右移|  

|数据运算符|描述|
|:---:|:---:|
|sizeof()|获取该数据的大小|
|[]|数组下标|
|&|取一个地址|
|\*|对地址取值|
|->|结构体解引用|
|.|结构体引用|  

|杂项运算符|描述|
|:---:|:---:|
|,|逗号运算符|
|()|括号运算符|  

|赋值运算符|描述|
|:---:|:---:|
|=|赋值|
|+=|加后赋值|
|-=|减后赋值|
|\*=|乘后赋值|
|/=|除后赋值|
|%=|取模后赋值|
|<<=|按位左移后赋值|
|>>=|按位右移后赋值|
|&=|按位与后赋值|
|^=|按位异或后赋值|
|!=|按位或后赋值|
